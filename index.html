Folder CRUD

Form içərisində btnOpen klik olunan zaman folderBrowserDialog-dan seçilən hər hansisa directory-də olan folder və ya file-ların dataGrid-də göstərilməsi
dataGrid-də 2 column olacaq Name , Type directory-də folder olarsa name hissədə adı type hissədə Folder deyə qeyd ediləcək, file olarsa Name hissəde fileName Type hissəsində File , hər ikisi üçün ad textBox vasitəsi ilə əlavə ediləcək :
1 btnCreateFile klik olunan zaman eyni directory-də istənilən uzantılı bir file yaradılır və eyni anda dataGrid-də elavə edilir
2 btnCreateFolder klik olunan zaman eyni directory-də folder yaradılır və eyni anda dataGrid-də əlavə edilir
Datagrid-də uzantısı .html, .css, .js, .txt olan file olan sətrlərdən her hansısa birinə klik olunan zaman müvafiq uzantıya sahib file-in içərisindəki data richTextBoxda göstəriləcək,(Məsələn: .html uzantili bir file göstərilirsə düzgün formatlandırın görünüşü)
Richtextbox-da dəyişiklik olunarsa btnSave aktiv olur ve olunan dəyişikliklər yadda saxlanır.
